# Status

## Example Status Object

```
{
  "code": 200,
  "message": "",
  "status": "",
  "data": "",
  "errors": [
    {
      "fields": [

      ],
      "messages": [

      ]
    }
  ]
}
```

* `code` (int64): Status http code
* `message` (string): Error message
* `status` (string): Status message
* `data`: Additional data
* `errors` (array): A list of api errors
