# ReleasePackage

## Example ReleasePackage Object

```
{
  "package_link": "https://www.files.com/desktop/download",
  "platform": "win32",
  "architecture": "amd64",
  "extension": "deb",
  "sha256": "example"
}
```

* `package_link` (string): Link to the package
* `platform` (string): Release platform
* `architecture` (string): Release architecture
* `extension` (string): Package file extension
* `sha256` (string): SHA256 hash of the package
