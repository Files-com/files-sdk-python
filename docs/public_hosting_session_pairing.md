# PublicHostingSessionPairing

## Example PublicHostingSessionPairing Object

```
{
  "pairing_key": "example",
  "redirect_uri": "example",
  "expires_at": "2000-01-01T01:00:00Z"
}
```

* `pairing_key` (string): One-time pairing key for the Public Hosting session
* `redirect_uri` (string): Public Hosting URL that exchanges the pairing key for a Public Hosting session cookie
* `expires_at` (date-time): When the pairing key expires
