# ShareGroupMember

## Example ShareGroupMember Object

```
{
  "name": "John Doe",
  "company": "Acme Ltd",
  "email": "johndoe@gmail.com"
}
```

* `name` (string): Name of the share group member
* `company` (string): Company of the share group member
* `email` (string): Email of the share group member
