# SyncApiUsageSnapshot

## Example SyncApiUsageSnapshot Object

```
{
  "ifname": "example",
  "api_reads": 1,
  "api_writes": 1,
  "list_cache_hits": 1,
  "list_cache_misses": 1,
  "stat_cache_hits": 1,
  "stat_cache_misses": 1
}
```

* `ifname` (string): Interface Name
* `api_reads` (int64): API Reads
* `api_writes` (int64): API Writes
* `list_cache_hits` (int64): List Cache Hits
* `list_cache_misses` (int64): List Cache Misses
* `stat_cache_hits` (int64): Stat Cache Hits
* `stat_cache_misses` (int64): Stat Cache Misses
